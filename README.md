# BBP Base Planner

A 3D base planner for the [DayZ BaseBuildingPlus (BBP)](https://steamcommunity.com/sharedfiles/filedetails/?id=1710977250) mod.

Plan your base, count your materials, and share your design — all in the browser.

## Features (in progress)

- 3D drag-and-place base building
- Full BBP parts catalog with tier upgrades
- Live material and resource counter
- Shareable URLs
- AI co-pilot (bring your own API key)

## Development

```bash
npm install
npm run dev
```
